# mf-js-form

## ES6

```javascript
import form_manager from 'mf-js-form';

const manager    = form_manager.manager;
const rules      = form_manager.rules;
const validators = form_manager.validators;
```

## CommonJS
```javascript
const form_manager = require('mf-js-form');

const manager    = form_manager.manager;
const rules      = form_manager.rules;
const validators = form_manager.validators;
```
